# XP-miner
It is a powerfull tool used to collect Xp in a creative island (code = 8442-8238-14520).

## How it works 
It can prevent the user from being kicked out by server for inactivity. It switches desktop to perform some movement and firing the gun few times to prevent being flagged as AFK by server.
This program runs fortnite in backround and switches to it every 5 min to maintain activity.

## Requirments
* Python 3
* Fortnite
* IDE (eg VS Code)
* Libraries-
  * pyautogui
  * keyboard
## Setup
Setup for this program is very simple.
Steps -
* open the program in an IDE in an desktop
* open fortnite game in the immidiate right desktop
* join a private game in the map (8442-8238-1452)
* equip a gun with good fire rate
* enter the secret room
* run the script and leave your system idle for some time

## Rates
This setup is capable of gathering a large amount of Xp in short time at a rate of **1.8 Levels per hour**.
